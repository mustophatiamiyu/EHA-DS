# Spacing System

There are 17 spacing rules, all based on multiples of four. These rules should be applied to the gaps between components, and it is advisable to follow them within components when feasible.

---

## Spacing Scale

All spacing tokens follow a predictable pattern based on multiples of 4px. This creates a cohesive visual rhythm across all components.

## Code Usage

### CSS Variables
```css
:root {
  --spacing-1: 0.25rem;   /* 4px */
  --spacing-2: 0.5rem;    /* 8px */
  --spacing-3: 0.75rem;   /* 12px */
  --spacing-4: 1rem;      /* 16px */
  --spacing-5: 1.25rem;   /* 20px */
  --spacing-6: 1.5rem;    /* 24px */
  --spacing-8: 2rem;      /* 32px */
  --spacing-10: 2.5rem;   /* 40px */
  --spacing-12: 3rem;     /* 48px */
  --spacing-16: 4rem;     /* 64px */
  --spacing-20: 5rem;     /* 80px */
  --spacing-24: 6rem;     /* 96px */
  --spacing-32: 8rem;     /* 128px */
  --spacing-40: 10rem;    /* 160px */
  --spacing-48: 12rem;    /* 192px */
  --spacing-56: 14rem;    /* 224px */
  --spacing-64: 16rem;    /* 256px */
}

/* Usage */
.card {
  padding: var(--spacing-6);
  margin-bottom: var(--spacing-8);
}

.button-group {
  gap: var(--spacing-2);
}
```
